# Projet : Analyse de séquences ADN

## Objectif
Lire un fichier FASTA, calculer le pourcentage de GC pour chaque séquence et visualiser la distribution.

## Outils utilisés
- Python
- BioPython
- Matplotlib
- Google Colab

## Résultats
- 5 séquences analysées
- Calcul du %GC pour chaque séquence
- Histogramme de la distribution du %GC
